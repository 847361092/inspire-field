====================================
项目依赖说明 Requirements
====================================

本项目是基于 Node.js 的 Web 应用，使用 npm 管理依赖。
所有依赖已在 package.json 中定义，无需额外安装。

一、系统要求
------------------------------------
Node.js >= 20.19.0 或 >= 22.12.0
npm >= 10.0.0（随 Node.js 自动安装）


二、依赖安装方法
------------------------------------

标准安装（推荐）：
npm install

使用淘宝镜像（国内用户推荐）：
npm install --registry https://registry.npmmirror.com

强制重新安装：
rm -rf node_modules package-lock.json
npm install


三、主要依赖包
------------------------------------

前端框架：
- vue@3.5.18                 # Vue3 框架
- vue-router@4.5.1          # 路由管理
- pinia@3.0.3               # 状态管理

构建工具：
- vite@7.0.6                # 构建工具
- typescript@5.8.0          # TypeScript
- @vitejs/plugin-vue@6.0.1  # Vue3 插件

动画库：
- animejs@4.1.2             # 动画库
- gsap@3.13.0               # 高级动画库
- lenis@1.3.8               # 平滑滚动

工具库：
- @vueuse/core@13.6.0       # Vue组合式API工具
- axios@1.6.2               # HTTP请求库
- marked@16.1.2             # Markdown解析

后端依赖：
- express@4.18.2            # Web框架
- multer@1.4.5-lts.1       # 文件上传
- cors@2.8.5               # 跨域支持
- formidable@3.5.4         # 表单解析


四、开发依赖
------------------------------------
- @types/node@22.16.5       # Node.js类型定义
- vue-tsc@3.0.4            # Vue TypeScript编译器
- npm-run-all2@8.0.4       # 并行运行脚本


五、快速开始
------------------------------------

1. 检查 Node.js 版本：
   node -v

2. 安装依赖：
   npm install

3. 启动项目：
   npm run dev:all

4. 访问：
   http://localhost:5173


六、故障排除
------------------------------------

如果 npm install 失败：

1. 清理缓存：
   npm cache clean --force

2. 删除旧文件：
   删除 node_modules 文件夹
   删除 package-lock.json 文件

3. 重新安装：
   npm install

4. 使用其他镜像源：
   npm install --registry https://registry.npmjs.org/


七、离线安装
------------------------------------

如果没有网络，需要：
1. 在有网络的电脑上运行 npm install
2. 将整个 node_modules 文件夹复制到目标电脑
3. 直接运行 npm run dev:all 即可
